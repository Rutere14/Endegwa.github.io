Tableau Project

This project focuses on developing an interactive dashboard for the Ministry of Agricultural Expansion in India, designed to monitor and evaluate the performance of four key programs: Samruddhi, Vikas, Jeevan, and Sarvang, across various states. Using a synthetic dataset spanning June to November 2022, the dashboard showcases critical metrics like beneficiaries, disbursements, loans, training sessions, new businesses supported, and funding trends. With a user-centric approach, the dashboard enables stakeholders to explore data dynamically, compare state-level performance, analyze disbursement versus funding, and gain actionable insights to assess program effectiveness.
